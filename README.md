## Implementation

1. MVC Architecture
2. State Management setState
3. Firebase
4. Authentication Login by Google
5. Local Storage Hive
6. Go To WhatsApp with url_launcher
7. Implementation Chart with syncfusion_flutter_charts
8. Deploy to Google Play Store
